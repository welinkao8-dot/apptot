# TOT FROTA Landing Page

Esta é a landing page oficial do **TOT FROTA**, um sistema moderno de gestão de frotas e rastreamento feito em Angola.

## Tecnologias
- React 19
- Vite
- Lucide React (Ícones)
- React Router Dom (Navegação)
- CSS Vanilla (Design Premium)

## Funcionalidades
- ✅ Hero Section com estatísticas e imagem realista
- ✅ Seção de Funcionalidades (Rastreio, Combustível, Segurança)
- ✅ Casos de Uso (Empresas, Pais, Indivíduos, Equipamentos)
- ✅ Planos de Preços
- ✅ Centro de Suporte com formulário
- ✅ Totalmente Responsivo

## Como rodar localmente
1. Entre na pasta: `cd landing-frota`
2. Instale as dependências: `npm install`
3. Inicie o servidor: `npm run dev`

## Deploy no Coolify
Este projeto já contém um `Dockerfile` configurado. Para adicionar ao Coolify:
1. Aponte para este repositório/pasta.
2. O Coolify detectará automaticamente o `Dockerfile`.
3. Certifique-se de que a porta 80 está mapeada.
